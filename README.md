Objective: Build a simple GUI to benchmark the PC using Geekbench and display the results.
Key Requirements:
•	GUI Features:
o	A button to "Run Test."
o	A text area or panel to display the progress and results.
•	Backend Integration:
o	Use the Runtime or ProcessBuilder class in Java to execute the Geekbench CLI (Command Line Interface) if available.
o	Parse the CLI output and present the scores (CPU, GPU, memory, etc.) in the GUI.
•	Required Skills:
o	Knowledge of Swing, JavaFX, or any GUI framework.
o	Ability to handle external process execution and output parsing.
